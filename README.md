# nodecg-twitter-widget
NodeCG layout bundle for showing tweet.

## Requirements

- `nodecg ^1.1.1`

## Installation

```
git clone https://github.com/cma2819/nodecg-twitter-widget.git
```

or

```
git install cma2819/nodecg-twitter-widget
```
